angular_todo
============

## Getting Started

1. Clone the repo
	
	`git clone`
	
2. Install dependencies

	`npm install && bower install`
	
3. Build the project

	`grunt build`
	
4. Run tests

	`grunt test`
	
5. Start server

	`grunt serve`
	
## About

This is a Yeoman generated Angular Todo App.

A live version can be found here:

[http://jpark3000.github.io/angular_todo](http://jpark3000.github.io/angular_todo)
